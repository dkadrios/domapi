Introduction

Author: Darin Kadrioski
Modified: 09/19/2004

Welcome to the world of DomAPI!

People use DomAPI for a variety of things. Some people just use it to manage dynamic content and some use it only for animations and effects. Some make full blown applications, and some only use it for form validation, or processing URL parameters.

The great thing about DomAPI is that these are all valid uses. There is no right or wrong way to use it. It is as much a development platform as it is an API. By default it loads a small set of base services that constitute its core "engine". From there you are free to load as little or as many other extensions and components as you wish, each adding more functionality to your applcation. At no point does the Library try to get in your way, you are free to use it in any manner you wish.

The more you use DomAPI, the more uses you will find for it. At the end of the day, DomAPI is just there to make your development life easier.

This tutorial will evolve over time to explore various aspects and usage of this wonderful Library. The tutorial itself is open to the public and anyone is welcome to contribute. Send inquiries to support@domapi.com if you are interested in adding to this tutorial.
