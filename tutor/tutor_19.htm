Installing the Library

Author: Darin Kadrioski
Modified: 09/19/2004

To get started working with the DomAPI, download and unzip the latest version into a directory on your web site. DomAPI requires no installation or special server settings. You can even install it to your local development machine if that's your game.
I would recommend installing it in your /scripts directory, but it really doesn't matter where you put it, just so long as your DHTML pages can create a relative path to it.

Once unzipped, verify your installation by trying out the files under /tests. They should work exactly like they do on this site. If not, verify that you kept the embedded directory structure intact while unziping the file.
