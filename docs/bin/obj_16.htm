domapi.Datepicker

See also: Elm
See also: Component

Allows the user to select a date using a calendar or by typing one in. Supports any date format and automatically validates user entry.
Datepicker is a descendant of Dropdown, and therefore inherits all its members, as well as uses its constructor parameters. Its dropDown is a Calendar component.

The user is free to edit the date displayed, or select a new one using the calendar control.
The calendar is automatically updated as the user types in the edit control. Internally, the control ensures that a vaild date has been entered.

The user can also enter 'fuzzy' input as well. This consists of keywords such as 'tomorrow', 'sunday', 'next month', etc..
The full range of fuzzy inputs available is defined by the lang file used.
