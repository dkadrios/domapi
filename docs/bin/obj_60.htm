domapi.Label

See also: Elm
See also: Component

This themeable text container allows you to align it's content both horizontally and vertically.

The label component can be thought of as a DIV that supports vertical as well as horizontal alignment.

Constructor Parameters unique to domapi.Label

ParameterTypeDefaultDescription
valueString  Initial text to display
