domapi.Spinedit

See also: Elm
See also: Component

Useful for allowing the user to select a whole number. They can either type in a number or use the spin buttons to modify it. Supports defined ranges and automatically validates user info.

 

Constructor Parameters unique to domapi.Spinedit

ParameterTypeDefaultDescription
doWarningbooleantrue  Alerts the user if they type in a value which is out of the controls range.
holdIntervalinteger500  This is amount of time the user must hold down the mouse button before the arrows start revolving.
maxinteger1000  Maximum value the user is allowed to enter.
mininteger0  Minimum value the user is allowed to enter.
revolveIntervalinteger100  This is the amount of time between the arrows revolving when the mouse is held down and the holdInterval has been exceeded.
stepinteger1  Determines how much the buttons change the current value when clicked.
valueinteger0  The intial value.
You can also use Elm and Component constructor parameters when creating a domapi.Spinedit instance.
