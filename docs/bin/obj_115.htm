domapi.Colorpane

See also: Elm
See also: Component

Colorpane is used to select colors. It is based on the default MS Windows® color chooser. The user can select a color using the mouse, or can enter one directly in either HEX, RGB or HSL formats.

 

Constructor Parameters unique to domapi.Colorpane

ParameterTypeDefaultDescription
valueString#FFFFFF  The intial color .
You can also use Elm and Component constructor parameters when creating a domapi.Colorpane instance.

oncancelclick ()

Fires if the user clicks the 'Cancel' button. Your application can respond to this and take appropriate action.

onokclick ()

Fires when the user clicks the 'OK' button. The onchange event also fires at the same time, so either can be used.
