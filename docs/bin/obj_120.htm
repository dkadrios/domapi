domapi.Imageradiogroup

See also: Elm
See also: Component

An Imageradiogroup is a group of related Imageradiobuttons . It encapsulates it's radiobuttons and provides a caption and visible border.

 

Constructor Parameters unique to domapi.Imageradiogroup

ParameterTypeDefaultDescription
colsinteger1  The number of columns to arrange the radiobuttons into.
textString  The caption to display above the group.
You can also use Elm and Component constructor parameters when creating a domapi.Imageradiogroup instance.
