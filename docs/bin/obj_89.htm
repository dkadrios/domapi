domapi.Notebook

See also: Elm
See also: Component

A notebook is a collection of panes, of which only one is ever visible at any given time. The panes are HTMLElements, usually a DIV or an IFRAME. The panes are all the same size (the size of the Notebook component) and are managed by setting the Notebook's page index.

Notebooks are used by the Pagecontrol and Imagepagecontrol components, but they can also be used on their own. They are an excellent starting point for a 'wizard' type interface.

addPage( arg )

The "arg" parameter fed to this method is an inline Object. See below for accepted parameters.
Any parameters not listed here that you add may also be attached to any returned Object.

Adds a new page to the notebook.

Available "arg" Parameters:
ParameterTypeRequiredDefaultDescription
typeString DIV  The type of HTMLElement to create the page out of. Typical types are DIV (the default) and IFRAME.  
urlString   If type is IFRAME, then this is the url of the page to be loaded into it.  
delayLoadboolean false  If type is IFRAME, then this property delays loading the url until this notebook page is made visible.  
refreshPageboolean false  If type is IFRAME, then this causes the page to be refreshed each time it is shown.

assignElement( arg )

The "arg" parameter fed to this method is an inline Object. See below for accepted parameters.
Any parameters not listed here that you add may also be attached to any returned Object.

This is advanced way of adding a page. It allows you to take an existing HTMLElement and trasnfer it into the notebook. You can specify it's index in the notebook, or have it appended to the end of the pages collection. You can also have it overwrite an existing page.

Available "arg" Parameters:
ParameterTypeRequiredDefaultDescription
idStringY  The id of the existing element you want to transfer to the notebook.  
indexinteger   Where in the pages collection you want the new page added. If not specified, it is added onto the end.  
overwritebooleanYfalse  If set to true, any existing content in the page is overwritten. Only applies when setting the index to an existing page.
