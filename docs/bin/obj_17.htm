domapi.debug

 

The debug unit has powerful methods that are helpful in creating DomAPI pages and components. These are domapi.debug.dump_props(obj) and domapi.debug.dump_var(varname).
There are also methods for bringing up and interacting with the DomAPI Debug Console. This console allows you to evaluate expressions and property values at runtime.

This unit also has methods for profiling or timing code blocks. Results are outputed to the debug console via the domapi.debug.dump_var() method

dump_props ( object )

This method outputs all the properties of the given object to the DomAPI debug window. Values are sorted and grouped by whether or not they have a value or are null. Any properties that are themselves objects, are hyperlinked for navigation.

Example:
domapi.debug.dump_props(elm1);
