domapi.shadow

 

This object provides methods for adding visual shadows to onscreen elements. These shadows utilize alpha-transparency in order to make the shadows appear life-like.

To use this object in your pages, first include it's unit using the following:

domapi.loadUnit("shadow");

You can then attach a shadow to an element using something like:

domapi.shadow.dropShadow(getElm("div1"));

... where "div1" is an element with that as it's id

The dropShadow() method takes additional optional parameters as well. See below for more.

Methods
dropShadow ( element, size, color )

Creates a basic drop shadow.
The shadow is placed underneath the element defined in the first parameter. This does not need to be an Elm, but can be any HTMLElement.
The size parameter determines how many pixels out from the element the shadow appears to fall. The default is 4 if you do not pass in a value.
The color parameter determines the color of the shadow, and the default is #555555 (gray) if you do not pass one. The shadow will gradually fade as it moves away from the element.

Example:
var theElm = domapi.getElm("someID");
domapi.shadow.dropShadow(theElm, 5, "gray");
Parameters:
ParameterTypeRequiredDefaultDescription
elementHTMLElementY  Element to attach the shadow to  
sizeinteger 4  Depth, or width, of shadow
