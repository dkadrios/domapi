domapi.reflow

Reflow allows elms to be resized with the same ratio as the parent it is attached to. The parent can be another elm or the window. Offset boundaries can be set to make sure that a dimensions doesn't change beyond a set limit. You can make an elm reflow in two manners: elm1=domapi.Elm({x:0,y:100,w:50,h:20,bgcolor:'blue'}); elm2=domapi.Elm({x:0,y:0,w:200,h:60,bgcolor:"blue"}); elm2.reflowOn(); //attaches elm2 to the window boundaries can be assigned at reflow ininiatlisation by passing a boundary object or later using the reflowSetBoundary function: elm1=domapi.Elm({x:500,y:500,w:10,h:10,bgcolor:"blue"}); elm1.reflowOn({maxY:10;minY:10}); elm1.setReflowBoundary({maxX:10;minX:10});

reflowOn ()

Turns reflow on for an elm. The parent and boundary object are specified as paremeters. The boundary object allows the min or max offset to be set that dictates how much a dimension may change by.
