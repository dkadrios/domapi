domapi.Colorpicker

See also: Elm
See also: Component

Provides the methods for a user to select a color using an instance of the Colorpane component in a dropdown.

The user is free to edit the color displayed, or select a new one using the drop-down controls.

This component inherits all the members of its parent class Dropdown.
