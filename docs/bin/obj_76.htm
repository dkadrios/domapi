domapi.align

Contains functions to align a selection of elms in many different configurations. The target elms are selected using selection.js are held in domapi.selection.elms

alignVertical ()

aligns the elms in domapi.selection.elms so that each elm is put vertically next to the previous elm in the array (top edge to bottom edge). The parameter determines the direction of alignment from the first elm.

back to top
centreHorizontal ()

aligns the elms in domapi.selection.elms so that their centres are the same horizontal distance from each other. The outer elms do no move.
