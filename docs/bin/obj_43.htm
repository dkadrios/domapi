sysutils.validate

 

This unit automatically loads the sysutils unit.
validate.js contains a set of form element validation routines.
All of the validation routines take a pointer to a form element as a parameter. Some also take the "description" of the field for use in error messages. For instance validate.string() is used to make sure a required text element has a non-empty value. I could pass it "last name" as the field description, and if the user hasn't filled in the field, a message saying "A valid last name is required" would appear.
Each validation routine returns true or false depending on whether the field value is ok. In the case where the result is false, the method automatically focuses the offending control and displays an error message to the user.
