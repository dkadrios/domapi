domapi.Imagecheckbox

See also: Elm
See also: Component

The Imagecheckbox is similar to a standard form element, except that the control uses a graphic and therefore looks the same on any platform. The default graphic used is defined by the theme, or you can create your own custom images.
Another difference between a standard form element, is that the caption displayed is part of the control too, that way the checkbox can be checked just by clicking the caption.

If you use an onclick event to respond to the user clicking the control, be sure and check its enabled boolean property. The onclick event will fire even if the control is disabled.
