domapi.Splitter

See also: Elm
See also: Component

Splitter is used to provide a visible divider between one or more elements, that the user can drag to cause them to grow or shrink. The items can also be made to simple move with the divider rather than resize too.

Splitter has just one main method, dependant which is responsible for associating elements with itself. These elements must be either Elms or Components .
