domapi.Rollover

See also: Elm
See also: Component

Building on the power of the Imagelist component, Rollover encapsulates a complete image rollover without the need for messy preloading or image caching.

Rollover uses a 3 or 4 position Imagelist. The four positions are

  1. normal
  2. over
  3. down
  4. disabled
The forth one is optional.

Constructor Parameters unique to domapi.Rollover

ParameterTypeDefaultDescription
countinteger3  Number of images in the strip.
hinteger  The height of a single image, the visual height of the control.
orientationStringhorizontal  Whether the images in the strip are arranged horizonatally or vertically. Has no effect on the visual presentation.
srcString  The image src.
winteger  The width of a single image, the visual width of the control.
You can also use Elm and Component constructor parameters when creating a domapi.Rollover instance.

Events
onclick ()

Fires when the user clicks the control. Besure and check the enabled boolean before taking any action, as the onclick event will fire even if the control is disabled.

Example:
rollover1.onclick = function(){
  if(this.enabled)print();
};
