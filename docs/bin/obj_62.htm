domapi.Tabset

See also: Elm
See also: Component

A Tabset is a collection of tabs, arranged in a row. These tabs can be oriented to be either at the top or the bottom. They also have a built-in 'feathering' value, for allowing it to be 'snapped' onto another control, such as a Notebook. This is exactly how Pagecontrol is created.

Each tab can have an optional reference to an HTMLElement, referred to as a 'pane'. When these references are present (passed in addTab) then the control will hide/show those panes based on which tab is selected. So if tab#2 has "someDiv" as its pane, "someDiv" will be shown whenever that tab is selected, and hidden when it is not.

Tabs automatically size themselves based on their captions. When there is not enough space for all the tabs to fit on a row, additional rows are added automatically.

Methods
addTab( arg )

The "arg" parameter fed to this method is an inline Object. See below for accepted parameters.
Any parameters not listed here that you add may also be attached to any returned Object.

Adds a new tab to the control. Tabs are added onto the end of the tabs array. Each tab has an index parameter, which is its index in this array. They also have a selected boolean and a pane property, which is a referenece to an element that this control should hide or show when this tab is selected.

Available "arg" Parameters:
ParameterTypeRequiredDefaultDescription
valueStringY  The caption for the tab. You can also use text.  
paneHTMLElement   If you want the control to automatically manage panes, pass a reference to one. The control will display and hide this pane based on when this tab is selected.
