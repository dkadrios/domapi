domapi.keyboard

The keyboard unit allows the user to test if a key is currently pressed or to attach events in a variety of ways. The current pressed keys are stored in an array, allowing the user to test for multiple key presses. There are two built in events that the user can attach a function to - onKeyUp and onKeyDown. the example below illustrates how to attach a function to onKeyDown: domapi.keyboard.onKeyDown = function() {alert('you just pressed ' + domapi.keyboard.lastKey);}; To remove the event assign it to null: domapi.keyboard.onKeyDown = null; You can also attach events to fire when a particular key is pressed: onK = function() {alert('you just pressed the K key');}; domapi.keyboard.attachEvent('k', onK); Keyboard events can also be removed as shown below: domapi.keyboard.removeEvent('k', onK);
