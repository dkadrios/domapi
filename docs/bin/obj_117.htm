Imagelistlite is identical to Imagelist except that it takes up fewer DOM resources. That is, it uses fewer onscreen elements to present itself. The downside to using Imagelistlite is that you have make sure your webserver doesn't cause IE's 'cache flicker bug' to occur.

See Imagelist for more info. It's identical to Imagelistlite..
