domapi.Imageradiobutton

See also: Elm
See also: Component

An Imageradiobutton is designed to behave similarily to a standard radiobutton in a form. The difference is that an Imageradiobutton is completely graphical in nature, so you can control the exact look of the element, regardless of platform or browser.

Imageradiobuttons can also be used together to form an Imageradiogroup.

If you use an onclick event to respond to the user clicking the control, be sure and check its enabled boolean property. The onclick event will fire even if the control is disabled.
