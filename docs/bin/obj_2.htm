domapi.Barchart

See also: Elm
See also: Component

Modeled after Microsoft Excel's® barchart, this component allows you to include a fully customizable barchart into your page without the need for Java or ActiveX, just pure DHTML goodness.

The barchart is broken up into several sections which can be independantly assigned, set, and controlled. The primary section is the chart section, created with the addChart() method (ie. addChart(x,y,w,h,gridlines) ) where gridlines is the number of gridlines desired (0 for none, of course). Gridlines (setGrid), Labels (addLabels), Names of the bars (setNames), chart Title (setTitle), and axis titles (setXAxis and setYAxis) can all be defined independent of each other by the respective methods mentioned.

Data can be added to the chart using the addData() method by including delimited lists of name/numeric-value combinations. An example of adding data to the barchart is
addData("12,20,32,40","Toast,Pancakes,Muffins,Eggs");

Various methods for defining and controlling all of the pieces and properties of the chart can be witnessed in the barchart example.
