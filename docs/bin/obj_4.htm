domapi.Button

See also: Elm
See also: Component

Not rocket science, just a simple button. What makes this viable however is that it matches the other component's look-and-feel and the application's Theme, so it fits well on your page.

The syntax to create a button is: 

button1= domapi.Button({text:"button1"});
After creation, the caption can be changed using the setText(caption) method. The caption itself can be simple text, or any valid HTML. You can use this fact to control the appearence of the button and to add icons, etc..

To respond to the user pressing the button, simply assign a method to the onclick() event.
Example:
<script src="../src/domapi.js"></script>
<script>
  domapi.loadUnit("button");
  onload = function(){
    button1 = domapi.Button({x:50,y:50,w:50,h:25,caption:"Button 1"});
    button1.onclick = button1_onclick;
  };
  function button1_onclick(){if(this.enabled)alert("onclick");};
</script>

Events
onclick ()

Fires after the user has clicked the button. Note that since this is a standard HTML event, you'll need to manually check if the button is enabled - it will fire regardless.

Example:
button1.onclick = function(E){
  if(this.enabled)
    alert("you clicked me!");
};
