domapi.Imagetabset

See also: Elm
See also: Component

Imagetabset is a tabset in which the tabs are completely graphical. There are no captions or custom drawing, the display is controlled completely through your images.

The tabs are Imagelists . They can be arranged either vertically or horizontally. The image lists have 5 positions:

  1. Selected
  2. Normal
  3. Over
  4. Down
  5. Disabled
Example:


The tabset can be arranged either horizontally or vertically. This arrangement is independant of the individual tab's arrangments, which is strictly for your own benefit when designing the images.

Imagetabset can automatically manage 'panes' associated with it. A 'pane' is just another HTMLElement that is tied to a certain tab. As that tab is selected and deselected, the 'pane' is shown or hidden using the method specified by the paneManage parameter.

Constructor Parameters unique to domapi.Imagetabset

ParameterTypeDefaultDescription
orientationStringhorizontal  Determines whether the tabset is vertical or horizontal.
overlapinteger0  Allows for the tabs to overlap. The z order is always left to right (or top to bottom if orientation is vertical), with the selected tab in the front.
paneManageStringvisibility  Either visibility or display. Determines how the component hides and shows any managed panes.
You can also use Elm and Component constructor parameters when creating a domapi.Imagetabset instance.
