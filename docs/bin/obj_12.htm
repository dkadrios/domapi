load ()

Parses any cookies located in document.cookie into the cookies object structure.
You must call this method after the page has loaded and before you try to access any of the cookies.

back to top
setValue ()

Sets the value of a cookie matching name or creates a new one if it didn't already exist.
Expiration of the cookie is controlled through the mins parameter. This is the number of minutes the cookie should stay alive. Default is 28 days.
