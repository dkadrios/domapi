domapi.cursors

 

domapi.cursors contains cross-browser string constants for popular CSS cursor properties. Use these to ensure browser compatability.

 

Properties unique to domapi.cursors

  • hand
  • horzBeam
  • vertBeam
Properties
handtype : String

This property contains the name of the cursor to use for the "pointing hand". The actual value varies from browser to browser. Use to ensure cross-browser conformance.

Example:
elm1.style.cursor = domapi.cursors.hand;
