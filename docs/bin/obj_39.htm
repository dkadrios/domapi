    formatString ( str, paramArray )

    If the passed string contains placeholders, then they are replaced by the params in the array, by ordinal value.
    Placeholders start with a percent sign and are followed by an integer. Only 1 through 9 are supported which means the array passed can have no more than 9 items.

    Example:
    sysutils.formatString("DomAPI %1 DHTML %2 easy!", ["makes","super"]);
// returns "DomAPI makes DHTML super easy!"

    ParameterTypeRequiredDefaultDescription

    defaultDateMasktype : Stringdefault value : mm/dd/yyyy

    For functions and components that display dates of use date formats, this is the mask used by default. Overwrite with your own locale's format.
    Accepts m,mm,d,dd,yy,yyyy and a single seperator type.

    Example:
    sysutils.defaultDateMask = "yy-mm-dd";

    monthLengthtype : Array

    Populated with the number of days in each month. Item[0] is January. February is listed as having 28 days. In years that are divisible by 4 or 400 (leap years), February should have 29. All functions in this unit perform this check automatically.

    Example:
    var daysInAugust = sysutils.monthLength[7];
